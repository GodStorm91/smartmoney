## Positioning: SmartMoney vs. Competitors

### Monarch (Target: Couples/Families, $14.99/mo)
- **Strength:** Best-in-class UX, comprehensive features
- **Weakness:** Cloud-only, no privacy guarantee, expensive
- **SmartMoney Play:** "Monarch power + your own server + privacy guarantee"

### Copilot (Target: iOS power users, $14.99/mo)
- **Strength:** Strong AI, iOS-native feel
- **Weakness:** iOS only, no household sharing, pricey
- **SmartMoney Play:** "Copilot AI + family sharing + works everywhere (web/Android/iOS)"

### Emma (Target: Financial health focus, $9.99/mo)
- **Strength:** Aggregation + insights, no-frills UX
- **Weakness:** Cloud-only, generic experience
- **SmartMoney Play:** "Emma insights + your choice of hosting + personalized AI"

### Honeydue (Target: Couples, free)
- **Strength:** Free, couples-focused
- **Weakness:** Limited features, no AI, basic visualization
- **SmartMoney Play:** "Honeydue features + AI assistant + gamification + premium options"

## Pricing Strategy

### Tier 1: Free
- Basic transactions, manual entry, categories
- Single user

### Tier 2: Premium ($9.99/mo)
- AI assistant, advanced visualizations, health score
- Receipt OCR, gamification
- Single or household (2 users)

### Tier 3: Family ($14.99/mo)
- Everything in Premium + unlimited household members
- Advanced family budget features
- Priority support

### Tier 4: Self-Hosted ($99 one-time or $9.99/mo)
- Full app (including premium features) deployed on user's server
- Updates + support included
- Option to fork + maintain custom deployment

### Tier 5: Enterprise (Custom)
- White-label, custom workflows, audit trails
- Regional data residency (Japan, Vietnam, EU)
- Compliance certifications (SOC 2, APPI, GDPR)
