## Risks & Mitigations

### High Risk: API Dependency (Plaid, Moneytree)
- **Mitigation:** Offer CSV import as fallback, cache aggressively, diversify providers
- **Impact:** If Plaid cuts access, users can still manually add transactions

### High Risk: LLM Cost Spiral
- **Mitigation:** Allow user-provided API keys, self-host option, on-device models
- **Impact:** User pays directly to LLM provider, SmartMoney stays lean

### Medium Risk: Regulatory Changes (Privacy)
- **Mitigation:** Monitor APPI/GDPR/CCPA quarterly, maintain compliance buffer
- **Impact:** May need feature adjustments, not core architecture changes

### Medium Risk: Competitor Feature Parity
- **Mitigation:** Differentiate on privacy + self-hosted, not feature list
- **Impact:** Focus on retention (gamification, UX) over acquisition
