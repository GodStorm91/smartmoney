## User Guide

### Generate Budget
1. Enter monthly income
2. AI generates budget allocation
3. Budget shows as "Draft" (not saved)

### Edit Budget
1. Swipe left/right on any allocation card
2. See real-time amount updates
3. Visual feedback guides you

### Save Budget
1. Click "Save Budget" button
2. See "Saving..." progress
3. Budget persists to database

### Regenerate Budget
1. Click "Regenerate" button
2. Optionally provide feedback
3. AI generates new draft
